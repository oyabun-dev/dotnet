# C# & .NET Learning Playground

This repository is dedicated to learning **C# basics** and exploring the **.NET ecosystem**. It serves as a personal workspace for experiments, tutorials, and small sample projects built while mastering the language and framework.

## 🎯 Objectives
- Understand core C# syntax, object-oriented principles, and advanced language features.
- Explore the .NET framework (e.g., console apps, libraries, ASP.NET Core).
- Practice testing, debugging, and adhering to C# best practices.

## 🚀 Getting Started

To compile and run the projects in this repository, ensure you have the following prerequisites installed:
- [.NET SDK](https://dotnet.microsoft.com/download) (Version 8.0 or latest recommended)
- A suitable IDE or text editor such as [Visual Studio Code](https://code.visualstudio.com/), [Visual Studio](https://visualstudio.microsoft.com/), or [JetBrains Rider](https://www.jetbrains.com/rider/).

### Basic CLI Commands

**Create a new console project:**
```bash
dotnet new console -n MyProject
```

**Run a project:**
Navigate to a project directory and execute:
```bash
dotnet run
```

**Build a project:**
```bash
dotnet build
```
